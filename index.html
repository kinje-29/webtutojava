Pengenalan Java

Java adalah bahasa pemrograman yang fenomenal. Sebagaian ahli mengkalim bahwa kelebihan-kelebihan pada bahasa pemrograman sebelumnya telah diadopsi oleh java. Dalam hal kepopuleran, menurut rilis TIOBE Index (www.tiobe.com) tiap bulan, java berada di peringkat dua besar (bergantian dengan C yang menjadi peringkat pertama) dalam tida tahun terakhir.

Java bisa digunakan pada berbagai keperluan. Pemrograman berbasis teks pada Clipper atau FoxPro for DOS yang bisa kita temui di kasir-kasir minimarket bisa dilakukan dengan java. Pemrograman berbasis GUI Desktop seperti yang dikenal pada Visual Basic, Visual FoxPro, atau Delphi juga bisa dilakukan dengan java. Pemrograman untuk mikro seperti ponsel atau tablet PC mudah dilakukan dengan java. Tidak ketinggalan, pemrogram web bisa menggunakan java Server Pages(JSP) atau Servlet. Anda mau membuat perangkat lunak untuk analisis, pengolah citra digital, game, atau utilitas. Java bisa menanganinya dengan sangat baik.

java Memiliki tiga edisi berbagai keperluan di atas, yaitu

  • Java 2 Standard Edition (J2SE) untuk pemrograman pada komputer Desktop.
  • Java 2 Microdevice Edition (J2ME) untuk pemrograman pada peranti mikro seperti ponsel.
  • Java 2 Enterprise Edition (J2EE) untuk membangun multi-tier enterprise seperti jaba bean dan Servlet.

Semuah edisi java tersebut bisa didapatkan secara geratis dan legal. Anda hanya perlu mengunduhnya dari situs resmi Java.com atau oracle.com. Java adalah sumber terbuka (Open Source) sehingga program berikut kode yang digunakan untuk pengembangan perangkat lunak bebas diunduh dan bebas di modifikasi. Kita bisa memakai Java di windows, linux, ataupun Sun Solaris. Digunakan di Macintosh pun tidak masalah. Dengan kata lain, java bisa digunakan pada berbagai platfrom (multipltfrom), sesuai dengan slogannya : "tulis program sekali, jalakan di mana saja" (Write once run anywhere).

Java juga mendukung fitur-fitur pengembangan perangkat lunak terbaru seperti object orienred programming (OOP). Java memiliki tingkat keamanan yang lebih baik dari kebanyakan bahasa pemrograman lain. Komunitas pengembangan java pun banyak. Bahkan untuk belajar java kita bisa jumpai di Internet. Hal-hal tersebut menyebabkan java terus berkembang.

Sedikit kelebihan java
  • Mengakomodasi kelebihan-kelebihan bahasa pemrograman sebelumnya.
  • Popularitas tinggi, hingga saat ini, berdasarkan TIOBE Index
  • Multiguna (bisa dipakai untuk berbagai keperluan): pemrogram berbasis teks, GUI (Desktop), pemrogram mobile, utilitas, grafis, game, web, DLL
  • Grafis dan sumber terbuka
  • Multipltfrom
  • Mengakomodasi fitur-fitur pengembangan perangkat lunak terbaru seperti OOP
  • Didukung Komunitas, referensi, dan pustaka (library) yang berlimpah.

Dalam bahsa pemrograman pasti ada kelebihan dan kekurangan masing-masing, demikian juga dengan java. Beberapa kekurangan java berikut bisa jadi bahan referensi bagi anda yang ingin belajar dalam bahasa pemrogram java, dibawah ini kekurang dalam bahasa pemrogram java.

  • Lama waktu pengeksekusian program relatif lebih tinggi dibandingkan dengan C. Hal ini disebabkan penerjemahan kode dilakukan lebih dari satu tingkat dengan adanya java Virtual Machine. Akan tetapi dengan adanya teknologi baru Just In Time (JIT) bisa mngurangi atau menambal celah tersebut.
  • Membutuhkan memori yang relatif lebih besar. Konsumen memori pada komputer cukup besar.
  • Mudah didekompilasi. Programer sulit menyembunyikan kode sumber yang telah dihasilkannya.
  • Pengodeannya relatif lebih sulit.
